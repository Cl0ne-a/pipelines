Описание:
Логично предположить, что вода в каналах может двигаться только в одном направлении:
в нашем случае, от меньшего к большему.

Соответственно, движение по линиям  1 ... 4, 1 ... 5, или 6 - 7.

При запуске com.protonmail.unit10 > Main, открывается схема водных линий. 
В консоли предлагается ввести пункт стартовый и пункт конечный, 
на выходе получаем расстояние между пунктами.
